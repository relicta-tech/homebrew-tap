# Homebrew Tap for Relicta

Official Homebrew tap for [Relicta](https://github.com/relicta-tech/relicta) - AI-powered release management for modern software teams.

## Installation

```bash
brew tap relicta-tech/tap
brew install relicta
```

Or install directly:

```bash
brew install relicta-tech/tap/relicta
```

## Upgrade

```bash
brew upgrade relicta
```

## Uninstall

```bash
brew uninstall relicta
brew untap relicta-tech/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `relicta` | AI-powered release management CLI |

## Documentation

See the [Relicta documentation](https://github.com/relicta-tech/relicta#readme) for usage instructions.
